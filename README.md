## Local Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/password-security-auditor.git
cd password-security-auditor
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
python app.py
```

4. Open browser to `http://127.0.0.1:5000`

## Tech Stack
- Python 3.11
- Flask
- JavaScript (Vanilla)
- HaveIBeenPwned API
